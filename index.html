出行攻略

山阳县

SHAANXI · 33.53°N 109.88°E
秦岭腹地 · 革命老区 · 山水秘境

陕东南的小城，藏在秦岭褶皱里。漫川关的明清老街、天竺山的云海日出、金钱河的清冽山泉，节奏比西安慢两档，适合周末和家人来一场"无目的"的深呼吸。

⛰️风景

  • ·天竺山
    国家 4A，秦岭云海、日出奇观，山顶有道教遗迹。
  • ·月亮洞
    天竺山脚下喀斯特溶洞，洞内钟乳千姿百态。
  • ·漫川古镇
    明清会馆建筑群，陕西南大门，骡帮商道遗韵。
  • ·苍龙岭森林公园
    森林覆盖率 95%+，夏季均温 22℃，避暑圣地。

🍜美食

  • ·山阳搅团
    玉米面打制，配蒜汁辣油，地道农家味。
  • ·腊肉炒粉条
    秦岭土猪腊肉，烟熏味浓，配红薯粉一绝。
  • ·橡子凉粉
    橡子粉手工制作，Q 弹爽滑，夏日解暑神器。
  • ·浆水面
    陕南家常，酸浆发酵醒神，配手擀面。
  • ·神仙凉粉
    糜子杆搓制，天然果冻感，配辣油醋汁。

西安市

SHAANXI · 34.27°N 108.95°E
十三朝古都 · 碳水之都 · 唐风汉韵

"一座长安城，半部中华史"。城墙根下的老茶馆、回民街的烟火气、大唐不夜城的霓虹——这座城既能在博物馆里硬刚千年，也能在凌晨两点的夜市里把你喂得走不动路。

🏯风景

  • ·兵马俑
    世界第八大奇迹，建议租电子讲解，3 小时打底。
  • ·西安城墙
    保存最完整的古城墙，骑行一圈 14km，城楼看落日。
  • ·大雁塔 / 大唐不夜城
    夜景天花板，免费演出和 NPC 沉浸互动。
  • ·华清宫 · 骊山
    《长恨歌》实景演出地，离市区 30km 一日游。
  • ·陕西历史博物馆
    何家村窖藏、兽首玛瑙杯，免费但要提前预约。

🥟美食

  • ·肉夹馍
    腊汁肉炖得入口即化，馍外酥内软，秦豫两派各有所长。
  • ·羊肉泡馍
    自己掰馍是关键，掰越小越入味，配糖蒜解腻。
  • ·biangbiang 面
    裤带宽面，辣油葱花，简单粗暴的碳水快乐。
  • ·凉皮 / 肉夹馍 / 冰峰
    "三秦套餐"，碳水 + 碳水的快乐组合。
  • ·葫芦鸡
    陕菜代表，皮酥肉嫩，西安饭庄和樊记最经典。
  • ·回民街 → 永兴坊
    游客选回民街，本地人选永兴坊，摔碗酒网红地。

北京市

BEIJING · 39.90°N 116.40°E
千年皇城 · 中轴线 · 胡同烟火

从紫禁城的红墙黄瓦到 CBD 的玻璃幕墙，这座城新旧交叠。清晨天安门广场升旗、傍晚什刹海胡同遛弯、深夜簋街撸串——每条街都有自己的一套时间表。

🏛️风景

  • ·故宫
    提前 7 天预约，钟表馆 / 珍宝馆强烈推荐加票。
  • ·慕田峪长城
    比八达岭人少景美，包车一日游或公交直达。
  • ·颐和园 · 圆明园
    皇家园林双壁，颐和园走长廊到佛香阁看昆明湖。
  • ·南锣鼓巷 / 什刹海
    胡同游，恭王府、烟袋斜街、后海酒吧街一线。
  • ·国家博物馆
    免费但要预约，后母戊鼎、孝端皇后凤冠必看。
  • ·798 艺术区
    老厂房改造的当代艺术地标，拍照打卡两相宜。

🦆美食

  • ·北京烤鸭
    全聚德 / 便宜坊老字号，大董 / 四季民福排队首选。
  • ·炸酱面
    老北京炸酱面馆遍地都是，菜码齐全才地道。
  • ·涮羊肉
    铜锅清汤 + 麻酱小料，东来顺、聚宝源常年排队。
  • ·卤煮火烧
    猪肠猪肺配死面饼，重口味深夜食堂代表。
  • ·豆汁儿 + 焦圈
    老北京"黑暗料理"，勇敢者的早餐。
  • ·簋街小龙虾
    麻辣、蒜蓉、油焖三派，麻小配啤酒的夏天标配。

成都市

SICHUAN · 30.57°N 104.07°E
天府之国 · 火锅之城 · 慢生活之都

"少不入川，老不出蜀"。宽窄巷子的盖碗茶、人民公园的掏耳朵、春熙路的时尚辣妹、青城山的清晨薄雾——成都把"舒服"两个字刻进了骨子里，来了就不想走。

🐼风景

  • ·大熊猫繁育研究基地
    7:30 开园最佳，看完月亮产房后从后山逛。
  • ·宽窄巷子 / 锦里
    宽窄巷子偏文创，锦里偏小吃，晚上灯笼亮起最出片。
  • ·武侯祠 / 杜甫草堂
    三国 + 诗圣两大 IP，红墙竹影打卡圣地。
  • ·都江堰 · 青城山
    一日游经典组合，世界文化遗产 + 道教名山。
  • ·春熙路 / 太古里
    现代成都核心，IFS 熊猫屁股、太古里裸眼 3D。
  • ·人民公园
    鹤鸣茶社喝盖碗茶、看老成都掏耳朵，市中心慢生活样本。

🌶️美食

  • ·火锅
    蜀大侠、小龙坎、电台巷火锅，鸳鸯锅救外地人。
  • ·串串香 / 钵钵鸡
    冷锅钵钵鸡（叶婆婆、藤椒味）比热锅串串更易上手。
  • ·麻婆豆腐
    陈麻婆豆腐百年老店，麻辣烫酥四大要素齐活。
  • ·夫妻肺片
    红油拌牛肉杂碎，下酒神器，廖老妈、盘飧市出名。
  • ·钟水饺 / 龙抄手 / 担担面
    成都小吃三件套，复制四川味道。
  • ·肥肠粉 + 锅盔
    甘记、白家高记，红薯粉 + 冒节子，配酥脆锅盔。
  • ·双流老妈兔头
    五香 / 麻辣两派，啃兔头配冰啤酒是地道夜宵。
